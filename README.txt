Para trabajar con este repositorio, necesitas tener Git Large File Storage (Git LFS) instalado en tu sistema. Git LFS se utiliza para manejar archivos grandes de manera más eficiente. 

Descargar Git LFS:

Visita la página oficial de Git LFS y descarga el instalador. https://git-lfs.com/

Una vez instalado, ejecuta el siguiente comando para configurar Git LFS en tu computadora:

git lfs install

Este comando solo necesita ser ejecutado una vez.

Ahora que tenes Git LFS instalado y configurado, podes clonar el repositorio normalmente, 

git clone https://github.com/camivaleff/TPMET4.git

Los archivos pesados estarán incluidos. La descarga puede tomar entre 3 y 10 minutos debido al peso de los archivos.
